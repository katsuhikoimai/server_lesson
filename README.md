#`ls`<br>
`ディレクトリ内に存在するファイルやディレクトリを一覧表示できる。`<br>`オプションとして、-a,-lがある。`<br>
`-a`<br>
`不可視ファイルも参照可能`<br>
`-l`<br>
`タイプや権限、所有者や所有グループなどの詳細表示`<br>
`今回の場合、`<br>
`ls/root/server_lesson/linux_practice`<br>
`のように参照可能。`<br>

#`cd`<br>
`パスを指定することで対応するディレクトリに移動することができる。`<br>
##`絶対パスと相対パス`<br>
-`絶対パス`<br>`最も上位の階層にある、ルートディレクトリから参照するパス。`<br>
-`相対パス`<br>`ワーキングディレクトリから参照するパス。`<br>

#`mkdir`<br>
`MakeDirectoryの略。文字通りディレクトリを作る。`<br>`オプションに-pを指定すると、途中に存在しないディレクトリも含めて作成してくれる。`<br>

#`touch`<br>
`ファイルを作成するコマンド。現在のワーキングディレクトリ直下に新規作成する。`<br>`あるいはパスを指定することでも作成可能。`<br>

#`&&`<br>
`AND演算子。左辺が真だった場合に右辺が実行される。`<br>

#`cp`<br>
`Copyの略。ファイルやディレクトリを複製する。`<br>`ディレクトリを複製する際は-rのオプションが必要。`<br>

#`mv`<br>
`moveの略。ファイルやディレクトリの移動を行う。`<br>`すでに同じファイル名が存在する場合、コマンドが実行されるとファイル名が変わる。`<br>

#`rm`<br>
`Removeの略。ファイルやディレクトリの削除を行う。`<br>`ディレクトリの削除の場合はrmdirとなるが、ディレクトリ内にファイルが入っていた場合はエラーが出て、削除ができないようになっている。まとめて削除したい場合はオプションの-rをつけると削除可能。`<br>

#`ワイルドカード`<br>
`ワイルドカードとは、任意の文字を支持するための特殊な文字記号のこと。対応する部分があるファイルに対してまとめて操作ができる。`<br>
-`*`<br>`*以降の文字が対応している場合コマンド操作可能。`<br>
-`?`<br>`指定した箇所一文字が同じファイルを選択し、コマンドを執行する。`<br>

#`cat`<br>
`指定したファイルの内容をターミナルに出力する。`<br>

#`>`<br>
`左辺の実行結果の出力を、右辺で指定し他ファイルに書き込む。`<br>`echo'HelloWorld>README.md'`<br>`となっていた場合、ReadmeファイルにHelloWorldが出力される。こちらは上書き処理になる。`<br>

#`>>`<br>
`左辺の実行出力結果を右辺で指定したファイルの末尾に書き加える。上記の>と合わせて、対応するファイルがなかった場合に新規作成する。上書きの心配なし。`<br>

#`vi`<br>
`ターミナルやコマンドプロンプト上で即時起動できるエディタ。ノーマルモード(編集をすることができないが、移動や保存、検索や削除ができる)とインサートモード(入力したキーがそのままファイルに反映され、編集可能)がある。`
